# Project 32: Employee Attrition Prediction (離職予測)

## 📌 概要
従業員の属性データ（年齢、勤続年数、勤務地など）から、離職（Stay/Left）の可能性を予測する分類タスクです。

## 🛠️ 実装力のポイント
- **データの数値化**: `get_dummies` や `map` を用いたカテゴリ変数の処理。
- **Stratified K-fold**: 実務で必須となる、データの偏りを考慮した交差検証の実装。
- **混同行列 (Confusion Matrix)**: 予測精度の内訳を分析し、モデルの弱点を把握する。

## 🚀 次のステップ
モデル構築後、どの特徴量が離職に強く寄与しているかを分析し、ビジネス的な洞察を得る。
